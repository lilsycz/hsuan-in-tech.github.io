# Jade · Interactive Web Demo Template

A minimal, no-build HTML/CSS/JS template you can deploy **for free** on GitHub Pages (or Netlify/Vercel).

## Quick start (GitHub Pages)
1. Create a new repo on GitHub, e.g. `jade-demo`.
2. Upload all files in this folder (index.html, style.css, script.js, /assets).
3. In repository Settings → Pages → Source: select **Deploy from a branch** and choose the `main` branch, `/root` folder.
4. Wait ~1 minute; your site will be live at `https://<your-username>.github.io/jade-demo/`.

## Customize
- Replace images in `assets/` and update text in `index.html`.
- Add more cards in the “Selected Work” grid.
- Tweak colors in `style.css` variables (top of the file).
- JS includes: theme toggle, scroll reveal, and a lightweight background animation.

## Optional: Netlify
- Drag & drop this folder to https://app.netlify.com/drop to get an instant link.
